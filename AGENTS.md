# 개요

이 저장소는 [오픈코드 (OpenCode)](https://opencode.ai/)를 사용하는 스펙 주도 자율 AI 워크플로를 지원합니다.

모든 명세는 [`specs/`](./specs) 디렉터리에 위치하며, 구현과 의사결정의 단일 진실 공급원 (source-of-truth)으로 취급해야 합니다.

코드 변경 전에는 관련 명세를 먼저 확인하고, 동작이나 요구사항이 바뀌면 해당 명세도 함께 갱신해야 합니다.

## 기타

* 매뉴얼, API 레퍼런스 등 관련 문서를 검색해야 할 때는 [Context7](https://context7.com/) MCP 도구를 사용해야 합니다.
* 소스의 모든 주석이나 로그 메시지, 그리고 테스트의 설명은 한글로 작성해야 합니다.
